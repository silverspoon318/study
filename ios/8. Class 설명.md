## Class and Structures
일반적인 목적은 빌딩 블록을 유연하게 만드는 역할을 한다.
property와 method를 정의한다.
class와 structure 를 single file로 정의해라.
new 연산자를 쓰지 않는다.

##차이점
class가 structure보다 기능이 좀더 더 있다.
중요. class는 상속이되고 형변환이 가능하다.

## Properties
값을 저장하기 위해서 쓰지만 저장을 안하는 property도 있다.

## Stored Properties
a stored property is a constant or variable

## Computed Properties
연산하는 property

## Property Observers
property 값이 변경하기 직전이나 직후나 추적을 할 수 있다.
